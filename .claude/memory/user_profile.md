# User Profile — 關於我

## 個人背景
- **神經特徵**：ADHD，需要外部腳手架來建立生活節奏
- **職業**：公司老闆，團隊不大
- **日常**：處理客戶、同事、財務、物流
- **位置**：住高雄，有辦公室
- **家庭**：有一個兒子，暱稱「豌豆」

## 工作環境
- **內部工具**：Google Workspace（Wiki、Sheets 管帳、連結）
- **庫存系統**：自有 ERP/POS/Google Sheets
- **物流**：黑貓、宅配通（無 API，手動查）
- **銀行**：網銀查帳（無 open banking API）
- **高頻中斷**：每天人問「錢到了沒」「貨出了沒」

## 溝通風格
- **直接簡潔**：不加稱號，不用贅詞
- **單線程**：一次討論一個主題，逐步推進
- **信息量**：不要資訊轟炸
- **輸入方式**：語音輸入多，偶爾會跑掉（需根據上下文判斷）
- **回應格式**：Telegram 優先，文字為主

## 決策模式
- **驅動力偏差**：容易被「不想浪費訂閱」驅動，需要踩煞車
- **開發陷阱**：容易陷入無限開發循環，需提醒「停止開發，開始使用」
- **工具好奇心**：對新工具高度好奇（多巴胺驅動），但真正需要的是穩定
- **優化觀點**：重視「夠用就好」而非「最佳化」

## 訂閱與工具
- **AI 訂閱**：Claude Pro、ChatGPT Plus、Gemini
- **開發環境**：德瑪（家）Mac mini M4 Pro，跑 OpenClaw 龍蝦系統
- **部署計劃**：小蝦（辦公室），未來部署客服 LINE@
- **常用軟件**：Claude Code、GPT Codex、Gemini App、Obsidian、Google Keep/Tasks

## AI 使用原則（已驗證）
1. **簡單事不要複雜解**
2. **能用現有工具的就不自己開發**
3. **龍蝦 = LINE 工作互動 + 生活遙控器**（Sonos、三星電視、客服）
4. **Claude Code = 個人認知夥伴 + 開發工作台**
5. **GPT Codex = 一次性腳本 + 自動化任務**
6. **Gemini App = 日常快速查詢 + Google 生態整合**
7. **停止開發，開始使用** ← 最重要

## 痛點清單（優先順序）
1. **知識管理**：需要 AI 賦能的知識庫（優先）
2. **倒腦系統**：語音→自動拆解成任務/行程/筆記
3. **持久記憶**：AI 記住風格、決策、context
4. **調用性**：一句話觸發複雜工作流
5. **生活助手**：提醒、排程、自動化日常

## 不要做的事
❌ 自動派工但不告訴我（透明度重要）
❌ 把我的記憶和龍蝦的記憶混在一起
❌ 一次資訊轟炸超過 3 個要點
❌ 用敬語或正式稱呼
❌ 假設我知道技術細節（要解釋）
