## ✅ 定案版本：v1.1.0 — 2026-03-24

> **這是用戶確認的最終規格，非必要不要更動。**
> 更動前必須先讀這段，說清楚為什麼要改。

### 定案規格（compose-poster.py）

**版面**
- 畫布：2160×3840（4K 直式，9:16）
- 背景：純黑 `(0, 0, 0)`
- 海報：未裁切，保持原始比例，contain 模式
- 海報寬度：畫布 95%（左右各 2.5% 黑邊）
- 海報位置：水平置中，底部對齊 InfoBar 上緣
- 外框：薄黑框 4px（圍繞海報外側）

**InfoBar（底部）**
- 高度：畫布高 8%，至少 160px
- 底色：純色深藍灰 `(14, 16, 22)`（不是半透明疊加）
- 標題字：STHeiti Medium → STHeiti Light → Georgia Bold（大寫）
- 副標字：STHeiti Light → Georgia → Arial Unicode
- 字色：標題 `(220, 220, 220)`，副標 `(150, 150, 150)`

**片名 / 導演語言原則**
- 一律用原文：英語片英文、日本片日文、韓國片韓文
- 使用 TMDB `original_title` 和 `original_language` 對應導演名

### 為什麼定案
- 用戶在 2026-03-24 確認這個效果「很好」，明確要求定案
- 之前多次修改版面（cover → contain、中文字體、外框）造成反覆，現在鎖定
