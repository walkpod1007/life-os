# Changelog

## v1.0.0 — 2026-03-23

### 新增
- 初始版本，透過 notebooklm-py CLI 查詢 NotebookLM 筆記本
- 支援 5 個主題筆記本路由（AI、科技、寶可夢、電影、性別）
- 登入帳號：walkpod@gmail.com

### 為什麼這樣做
- NotebookLM 個人版免費，基於已匯入來源回答，零幻覺
- 決策點：Enterprise $135/月太貴，個人版夠用
- notebooklm-py 是目前唯一能自動化查詢 NotebookLM 的開源方案

### 已知問題 / 待觀察
- notebooklm-py 登入狀態會過期，需定期 `notebooklm login`
- 5 個主題筆記本只有「AI 人工智慧」建好，其餘 4 個待建
- 查詢品質取決於已匯入的來源數量
