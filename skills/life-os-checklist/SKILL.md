# Life OS Checklist

## Overview

讀 `CLAUDE.local.md` 的週期事務清單，根據今天日期計算哪些該動了。

## 資料格式（在 CLAUDE.local.md 填寫）

```
事項 | 週期 | 上次執行 | 下次預計 | 備註
```

範例：
```
機油更換 | 每 6 個月 | 2025-12-01 | 2026-06-01 | 保養廠預約
牙科檢查 | 每年 | 2025-09-15 | 2026-09-15 | -
信用卡對帳 | 每月 | 2026-03-01 | 2026-04-01 | -
```

## 執行邏輯

1. 讀 `CLAUDE.local.md` 週期事務區塊
2. 計算每項的「距今天天數」
3. 分類輸出

## 輸出格式

```
📋 Life OS 週期事務 YYYY-MM-DD

🔴 過期（需立刻處理）
• 事項 — 上次：日期，已逾 N 天

🟡 本週到期
• 事項 — 預計：日期

🟢 本月內
• 事項 — 預計：日期（N 天後）
```

## 完成後更新

在 `CLAUDE.local.md` 更新「上次執行」和「下次預計」欄位。

## 分類

- 🚗 車輛：保養、輪胎、保險、驗車
- 🏥 健康：健檢、牙科、視力
- 💰 財務：報稅、對帳、月結
- 🏠 居家：濾網、家電保養
- 📦 消耗品：補充提醒
